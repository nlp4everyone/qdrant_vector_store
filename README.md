# 🛸 Introduction:

Qdrant Vector Service for enabling various RAG and searching techniques.

<br />

# 🐍 Python Version:

Tested with Python 3.9.

<br />

# 🔗 Installing:
- After cloning the project, install dependencies by commands:
```
pip install -r requirements.txt
```
